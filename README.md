# jepsen-spacetimedb

## Jepsen Tests for SpacetimeDB

Why SpacetimeDB?

- interesting architecture
- distributed sync
- dynamic community
- developers are responsive to interactions

SpacetimeDB's release of its version 2 and benchmarking tests have attracted attention and generated some controversy around its and other's consistency models, isolation levels and durability.

Lets develop a suite of Jepsen tests for SpacetimeDB to see what we can learn, and hopefully contribute back to the project.
